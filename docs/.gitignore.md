# .gitignore

## Purpose

Defines repository ignore rules for generated or local-only paths.

## Current Rules

- `assets/Tileset/`
- `assets/Tileset/**`
- `**/Tileset/`
- `**/Tileset/**`

## Impact

- Prevents Tileset folders and their contents from being tracked anywhere in the repository.
- Enforces a stable rule so Tileset assets are always excluded from commits.
